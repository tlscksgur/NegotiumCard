📇 Negotium Card

AI 기반 스마트 명함 관리 서비스

🚀 프로젝트 개요

Negotium Card는 종이 명함을 단순히 저장하는 수준을 넘어,
AI를 활용해 명함 정보를 자동으로 구조화하고,
비즈니스 네트워크를 탐색 가능한 데이터로 전환하는 서비스입니다.

명함 이미지를 업로드하면 이름, 회사, 부서, 직책, 이메일, 전화번호 등의 정보를 자동으로 추출하여
디지털 명함으로 변환하고, 이를 기반으로 인물 검색 및 조직도 탐색 기능까지 제공합니다.

💡 핵심 가치
📌 종이 명함 → 검색 가능한 데이터 자산으로 변환
📌 단순 저장 → 관계 기반 네트워크 관리
📌 개인 보관 → 구조화된 조직 정보 탐색
✨ 주요 기능
📷 명함 등록
이미지 업로드 또는 수기 입력 지원
다양한 형식의 명함 데이터 입력 가능
🤖 AI 기반 정보 추출
YOLO 객체 탐지 + OCR을 활용한 텍스트 추출
이름 / 회사 / 부서 / 직책 / 이메일 / 전화번호 자동 인식
📝 데이터 편집 및 관리
추출 결과 수정 및 보완 가능
디지털 명함 상세 정보 관리
🔍 인물 검색
이름 및 회사 기준 빠른 검색
누적된 명함 데이터를 효율적으로 탐색
🏢 조직도 탐색
회사 및 부서 구조 기반 시각화
인물 간 관계를 직관적으로 파악
🔐 개인 명함첩 관리
JWT 인증 기반 사용자별 데이터 관리
개인화된 명함 저장 및 조회
