명함 이미지를 AI로 분석해 연락처를 디지털 자산으로 변환하고, 인물 검색과 조직도 탐색까지 지원하는 스마트 명함 관리 서비스

프로젝트 설명
Negotium Card는 종이 명함을 단순 저장하는 데서 끝나지 않고, 업로드한 명함 이미지에서 이름·회사·부서·직책·이메일·전화번호를 자동 추출해 디지털 명함으로 정리하는 서비스입니다. 사용자는 명함을 이미지로 등록하거나 직접 수기 입력할 수 있고, 추출 결과를 수정한 뒤 인물 검색과 회사별 조직도 탐색으로 연결해 비즈니스 네트워크를 체계적으로 관리할 수 있습니다.

핵심 기능

명함 이미지 업로드 및 수기 등록
YOLO + OCR 기반 명함 정보 자동 추출
추출 결과 편집 및 디지털 명함 상세 보기
이름·회사 기준 인물 검색
회사/부서 구조를 반영한 조직도 탐색
JWT 인증 기반 개인 명함첩 관리
기술 스택 설명
프론트엔드는 React + TypeScript + Vite로 구현했고, 메인 백엔드는 Spring Boot + JPA + MySQL로 명함/인물/회사 데이터를 관리합니다. AI 분석 서버는 FastAPI 기반으로 동작하며, YOLO 객체 탐지와 Tesseract OCR을 조합해 명함 이미지에서 핵심 정보를 추출합니다.
