# 📇 Negotium Card

## AI 기반 스마트 명함 관리 서비스

---

## 🚀 프로젝트 개요

Negotium Card는 종이 명함을 단순히 저장하는 수준을 넘어,  
AI를 활용해 명함 정보를 자동으로 구조화하고 비즈니스 네트워크를 탐색 가능한 데이터로 전환하는 서비스입니다.

명함 이미지를 업로드하면 이름, 회사, 부서, 직책, 이메일, 전화번호 등의 정보를 자동으로 추출하여 디지털 명함으로 변환하고, 이를 기반으로 인물 검색 및 조직도 탐색 기능까지 제공합니다.

---

## 💡 핵심 가치

- 종이 명함을 검색 가능한 데이터 자산으로 변환
- 단순 저장을 넘어 관계 기반 네트워크 관리 지원
- 개인 명함첩을 구조화된 조직 정보 탐색으로 확장

---

## ✨ 주요 기능

### 📷 명함 등록

- 명함 이미지 업로드
- 직접 수기 입력 지원
- 다양한 형식의 명함 데이터 입력 가능

### 🤖 AI 기반 정보 추출

- YOLO 객체 탐지 기반 명함 영역 분석
- OCR을 활용한 텍스트 추출
- 이름, 회사, 부서, 직책, 이메일, 전화번호 자동 인식

### 📝 데이터 편집 및 관리

- AI 추출 결과 수정 및 보완
- 디지털 명함 상세 정보 조회
- 개인 명함첩 저장 및 관리

### 🔍 인물 검색

- 이름 기준 검색
- 회사 기준 검색
- 누적된 명함 데이터 탐색

### 🏢 조직도 탐색

- 회사 및 부서 구조 기반 조직도 제공
- 인물 간 관계를 직관적으로 확인

### 🔐 인증 기반 개인화

- JWT 인증 기반 로그인
- 사용자별 명함 데이터 관리

---

## ⚙️ 시스템 흐름

1. 사용자가 명함 이미지를 업로드합니다.
2. AI 서버에서 YOLO 객체 탐지와 OCR 텍스트 추출을 수행합니다.
3. 추출된 명함 정보를 백엔드로 전달합니다.
4. DB에 구조화된 데이터로 저장합니다.
5. 저장된 데이터를 기반으로 검색 및 조직도 기능을 제공합니다.

---

## 🎯 기대 효과

- 명함 정보를 단순 저장이 아닌 데이터 자산으로 활용
- 인물 및 조직 관계 기반 비즈니스 네트워크 관리
- 반복적인 수기 입력 최소화
- 사용자별 개인 명함첩 관리 효율 향상
